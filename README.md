## Personal Portfolio

This is a starter pack

### How to Deploy ?
1. Fork the Repo.
2. Head over to https://github.com/<  yourusername  >/MyPortfolio (STARTER branch)
3. Clone it in your system.
4. Now cd into the folder, then

```js
npm install
```
if required*
```js
npm audit fix --force
```

5. Deploy it on heroku.
